Memantau Perubahan Vegetasi
Merauke, 2023–2025

Merauke Vegetation Monitor mengolah data klasifikasi vegetasi berbasis citra Sentinel-2 dan metode Random Forest menjadi peta interaktif dan dashboard analitik — memuat area penyusutan (loss), penambahan (gain), dan area vegetasi stabil di seluruh cakupan analisis.

Tentang Proyek

Dashboard ini dikembangkan untuk menyajikan hasil analisis perubahan tutupan vegetasi di Kabupaten Merauke secara interaktif — dari peta spasial hingga ringkasan statistik.

Ruang Lingkup

  • Navigasi multi-modul: profil wilayah, program Food Estate, dataset, metodologi, peta interaktif, dan dashboard analitik.
  • Perenderan WebGIS dengan manajemen layer (basemap, kategori data, periode waktu).
  • Visualisasi grafik statistik dinamis berbasis data hasil klasifikasi vegetasi.

Teknologi

HTML5 / CSS3 / Vanilla JS Leaflet.js Chart.js Sentinel-2 Random Forest

Panduan Singkat

  • Gunakan menu navigasi di bagian atas untuk berpindah antar modul.
  • Pada modul WebGIS: centang/hilangkan centang layer di pojok kanan atas peta, klik poligon untuk detail, gunakan kolom pencarian untuk melompat ke lokasi.
  • Tombol ukur (penggaris) mengaktifkan mode pengukuran jarak — klik beberapa titik di peta, klik dua kali untuk selesai.
  • Pada modul Dashboard, ganti dropdown periode untuk memperbarui seluruh kartu metrik dan grafik.
  • Tombol Unduh Data di pojok kanan atas mengekspor statistik luas vegetasi (CSV).
