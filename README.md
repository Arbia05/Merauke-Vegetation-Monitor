# 🌿 Merauke Vegetation Monitor

**Merauke Vegetation Monitor** adalah web dashboard interaktif untuk memantau, menganalisis, dan memvisualisasikan kondisi vegetasi di wilayah Merauke, Papua Selatan.

---

## 🚀 Fitur Utama

- 📍 **Peta Interaktif:** Visualisasi spasial kawasan vegetasi di Merauke.
- 📊 **Dashboard Monitoring:** Tampilan data statistik dan kondisi vegetasi wilayah.
- ⚡ **Ringan & Aksesibel:** Berbasis teknologi web native sehingga dapat diakses langsung dari browser tanpa instalasi rumit.
